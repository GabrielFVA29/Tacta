<h1 align ="center"> TACTA </h1>
<h3> O TACTA é um jogo com proposta de inclusão para deficientes visuais, no qual o objetivo é sentir a vibração inicial e reconhecer a mesma vibração nas alternativas.</h3>

<h3> O TACTA é um projeto interativo no qual se utiliza de vibração e áudio para seu funcionamento.</h3>
<h2 align="center"><b>Como Executar🔨</b></h2>
<h3> Passo 1: Instalação do Arduino no Computador</h3>
<p> Para rodar o código, é necessário instalar o <a href="https://www.arduino.cc/en/software/">Arduino IDE</a> no seu dispositivo,<br> para isso, você pode apertar no hiperlink disponível nesse parágrafo.</p>
<p> Depois de acessar o link, aperte no botão de download e instale a versão mais recente disponível do programa.</p>
<p> Logo em seguida, abra o software instalado no seu gerenciador de arquivos e siga a instalação do modo que o programa sugere (Não mude nenhuma configuração)</p>
<p> Desse modo, o Arduino IDE será instalado no seu dispositivo!</p>
<h3> Passo 2: Integrar o código no circuito do arduino💻</h3>
<p> Para seguir o próximo passo, é necessário que o circuito esteja montado, caso não tenha o circuito em mãos, aqui está o <a href = "Artigo de tutorial do tacta.pdf">tutorial</a> para a montagem do TACTA.</p>
<p> Com o circuito montado, conecte o arduino com o computador pela entrada USB, em seguida, no aplcativo do arduino, selecione o arduino no canto superior esquerdo (especifique o modelo do arduino utilizado)</p>
<p> Depois de conectar o circuito com o computador, copie e cole o código presente nesse repositório, com o nome de <a href = "https://github.com/GabrielFVA29/Tacta/blob/main/tacta.ino">tacta.ino</a> dentro do arduino IDE</p>
<p> Caso esteja tudo pronto, aperte no botão da seta no canto superior esquerdo para mandar o código para o hardware executar.</p>
